# Literature Loop

## Purpose

Build a verified map of related work and citation-supported claims.

## Steps

1. PI Agent creates a literature task packet.
2. Literature Agent searches and extracts candidate sources.
3. Literature Agent verifies DOI/stable URL.
4. Literature Agent writes source summaries.
5. Reviewer 2 Agent critiques relevance and overclaiming.
6. PI Agent decides whether to add claims to the citation map.
7. Human approval required before manuscript-level literature claims.

## Output files

- `docs/citation_claim_map.csv`
- `docs/literature_notes/`
- `docs/claim_registry.md`
