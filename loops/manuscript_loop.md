# Manuscript Loop

## Purpose

Write the manuscript from verified project artifacts.

## Steps

1. PI Agent selects section and approved claims.
2. Manuscript Editor drafts prose.
3. Reviewer 2 Agent critiques overclaiming.
4. Literature Agent verifies citations.
5. Reproducibility Auditor checks that all numeric claims map to scripts/results.
6. PI Agent opens PR with section edits.

## Output files

- `manuscript/main.tex`
- `manuscript/supplement_proofs.tex`
- `docs/claim_registry.md`
