\begin{abstract}
Computational cognitive models are often compared by asking which model best fits behavioral data. This paper develops a formal and simulation-based argument that model-selection procedures can favor a more complex cognitive model when a simpler contextual model generated the data. The central concern is that unmodeled contextual heterogeneity can be misattributed to additional cognitive parameters. We formalize this problem, evaluate its consequences for Bayesian and predictive model comparison, and propose an embedded cognition framework in which cognitive parameters are modeled as context-sensitive psychological mechanisms.
\end{abstract}
