# Literature Verification Policy

## Rule

No literature claim may enter the manuscript unless it has traceable support.

## Required citation metadata

Each cited work should have:

- Authors.
- Year.
- Title.
- Journal/proceedings/book.
- DOI or stable URL.
- Verification status.
- Local PDF path or Zotero key if available.
- Exact claim supported.

## Uncertain references

If an agent is uncertain that a reference exists, it must mark it as `uncertain` and not use it in the manuscript.

## Claim-to-source discipline

Each paragraph in the literature review should map to at least one entry in `docs/citation_claim_map.csv`.
